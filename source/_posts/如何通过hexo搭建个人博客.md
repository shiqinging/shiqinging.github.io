# 前提条件

- node环境

- git

# 搭建过程

1. 创建一个本地文件夹

2. 在当前文件夹打开claude

3. 输入提示词

    ```Shell
    在当前目录下创建一个hexo静态博客，主题为fluid
    ```

然后根据步骤来，一步一步确认。

1. 输入提示词

    ```Shell
    所有内容都是中文的
    ```

然后根据步骤来，一步一步确认。

1. 输入提示词

    ```Shell
    导航区域也需要都是中文，导航区域的"关于"不需要了
    ```

然后根据步骤来，一步一步确认。

1. 输入提示词

    ```Shell
    打字机的内容为：精心整理的笔记与文档，让知识触手可及
    ```

      如果没看到效果，就说明改了配置文件，需要重启项目，让claude帮忙重启项目

2. 输入提示词

    ```Shell
    最终项目打包文件夹为docs，最后需要部署到：https://github.com/shiqinging/shiq
      inging.github.io.git  ，帮我调整一下配置，保证能在github page上访问
    ```

3. 输入提示词

    ```Shell
    页面中下面的版权部分不需要
    ```

4. 输入提示词

    ```Shell
    帮我重启项目
    ```

5. 输入提示词

    ```Shell
    copyright 也不需要，最后帮我重启项目
    ```

6. 打包代码

   ```shell
    帮我build一下项目
   ```

7. 通过可视化git提交代码
